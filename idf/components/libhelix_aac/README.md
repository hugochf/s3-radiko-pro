# libhelix_aac

Helix AAC decoder (fixed-point, C), vendored from
[pschatzmann/arduino-libhelix](https://github.com/pschatzmann/arduino-libhelix)
`src/libhelix-aac`.

**License:** RealNetworks Public Source License (RPSL/RCSL) — see the header block
in each source file. Open-source and source-available. This project is
learning-only and not distributed (see ../../PLAN.md).

Public API: `aacdec.h` (`AACInitDecoder`, `AACDecode`, `AACGetLastFrameInfo`,
`AACFindSyncWord`, `AACFreeDecoder`).
